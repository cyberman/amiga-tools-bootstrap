# amiga-tools-bootstrap

Diese Diskette enthält essentielle Shell-Tools für ein frisches AmigaOS 3.x System:

- lha (v2.15)
- unzip
- more

## Verwendung

1. Diskette in FS-UAE als DF0: einbinden
2. AmigaOS starten
3. Shell öffnen:
    ```amiga
    df0:
    installtools
    ```
4. Tools stehen nun systemweit unter `C:` zur Verfügung
